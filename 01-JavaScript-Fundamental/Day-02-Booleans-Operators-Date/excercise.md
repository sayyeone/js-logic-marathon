### no 5
```js
console.log(NaN === NaN) // false
console.log(null == undefined) // true
console.log(null === undefined) // false
console.log('' == false) // true
console.log('' === false) // false
```
1. false, karena value yg tidak dapat ditentukan maka hasilnya juga tidak valid maka false
2. true, karena nilainya sama" kosong
3, false, karena membandingkan tipe data dari null dan undefined tidak setara
4. true, karena string kosong merupakan falsy maka sama" dianggap false
5. false, karena tipe datanya berbeda
