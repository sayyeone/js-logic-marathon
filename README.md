## 📌 About This Repo

This is my log of learning JavaScript from the ground up. Instead of just watching tutorials and forgetting what I learned, I'm documenting everything here: the concepts, the exercises I worked through, the bugs I got stuck on, and the small projects that came out of it.

**Scope: JavaScript in depth fundamentals, core built-in objects (Sets, Maps, JSON, Regex), closures, classes, error handling, Promises/async-await, and DOM manipulation.** React and any backend/full-stack work come later, in a separate phase. This repo is about actually understanding JavaScript itself before building on top of a framework.

Topics are covered in a specific order, not randomly each one builds on concepts from earlier days.

Why bother writing all this down publicly:
- It keeps me honest about actually finishing what I start, one day at a time
- It's easier to see real progress over time than to just "feel" like I'm learning
- If someone ever asks what I've been studying, I can point here instead of saying "I watched some YouTube videos"

## 🗓 Learning Plan

**Part 1 JavaScript Fundamentals (Day 1–17)**
Variables & data types, booleans/operators/date, conditionals, arrays, loops, functions, objects, higher-order functions, Sets & Maps, destructuring & spreading, regular expressions, console methods, error handling, classes, JSON, web storage, closures.

**Part 2 Asynchronous JavaScript (Day 18–19)**
Promises, then async/await built on top of functions and error handling from Part 1.

**Part 3 DOM (Day 20–22)**
Selecting elements, manipulating the DOM, event listeners.

Each part only starts once the previous one is actually done working exercises and projects behind it, not just a fixed number of days passed. Day-by-day breakdown gets adjusted as I go; the tracker below reflects the current plan.
